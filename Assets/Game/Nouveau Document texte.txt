## Pathnote Skin Designer Beta v0.0.3

### Fixed Glitches
- You could use shortcuts when focused in an input field.
- When deleting a texture, the gun texture stayed equiped.

### New Features
- Allowed mutliple items selection when importing medias.
- Added a window where you can select all the textures in the media window.
Also allows you to remove/reset a window by selecting the item "None".

### Removed Features
- You can't select a texture on disk by clicking on it's texture holder anymore.
